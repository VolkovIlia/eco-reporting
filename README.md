# ЭкоОтчёт

SaaS для экологической отчётности российских предприятий.

## Описание

Автоматизация подготовки и подачи экологической отчётности в соответствии с требованиями Российской Федерации:

- **ПЭК** — Представление экологической информации
- **2-ТП (воздух/вода/отходы)** — Форма федерального государственного статистического наблюдения
- **НВОС** — Негативное воздействие на окружающую среду
- **Экосбор** — Вознаграждение за негативное воздействие

## Технологический стек

- Next.js 15
- TypeScript
- React
- PostgreSQL
